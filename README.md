## ğŸ“– Overview

**Sage Perfumes** is a front-end web project designed to simulate a high-end e-commerce experience. The project focuses on creating a sophisticated visual identity using deep gradients, elegant typography, and smooth user interactions.

Beyond the aesthetics, it features a fully functional **Vanilla JavaScript shopping cart**, dynamic product search, and responsive navigation.
